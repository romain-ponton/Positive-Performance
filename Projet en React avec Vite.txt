## 🌐 Architecture Application React – Positive Performances

### 1. **Routing Principal (`App.jsx`)**

```text
App
 ├─ AuthProvider (Context pour utilisateur connecté + rôle)
 ├─ Router (react-router-dom)
 │   ├─ /login → LoginForm
 │   ├─ /register → RegisterForm
 │   ├─ /home → HomePage
 │   ├─ /signature → SignaturePage
 │   ├─ /resources → ResourcesPage
 │   ├─ /events → EventsPage
 │   ├─ /education → EducationPage
 │   └─ /admin → AdminDashboard (Admin uniquement)
 └─ ProtectedRoute (vérifie rôle utilisateur)
```

---

### 2. **Pages Principales (`src/pages`)**

* **HomePage.jsx**

  * Composants : `SidebarMenu`, `ContentDisplay`
* **SignaturePage.jsx**

  * Menu déroulant : FR / EN
  * Sous-menus pour chaque formation
* **ResourcesPage.jsx**

  * Menu : Masterclasses / Self Inquiry Tools and Surveys
  * Permissions : `user_standard` bloque Self Inquiry Tools and Surveys
* **EventsPage.jsx**

  * Menu : Forbes Coaches Council / Featured on Forbes / Blog Article
* **EducationPage.jsx**

  * Menu : Masterclasses by Krumma and team
* **AdminDashboard.jsx**

  * Liste des utilisateurs
  * Gestion des pages et contenus
  * Statistiques / Dashboard

---

### 3. **Composants Réutilisables (`src/components`)**

* **SidebarMenu.jsx**

  * Props : `menuItems`, `onItemClick`
  * Affiche menus déroulants et sous-menus
* **ContentDisplay.jsx**

  * Props : `content` (texte, fichiers, vidéo)
  * Change dynamiquement selon menu sélectionné
* **VideoPlayer.jsx**

  * Lecture vidéo intégrée (mp4, YouTube, Vimeo…)
* **LoginForm.jsx / RegisterForm.jsx**

  * Formulaire d’authentification
* **ProtectedRoute.jsx**

  * Vérifie rôle utilisateur avant d’accéder à la page

---

### 4. **Context / State Management**

* **AuthContext**

  * `user` : { id, email, role }
  * `login()`, `logout()`, `register()`
* **ContentContext** (optionnel si contenu dynamique)

  * Contenus par page + menu
  * Permet à l’admin de modifier les contenus

---

### 5. **Gestion des Rôles et Permissions**

| Page / Contenu                           | user_standard | user_premium | admin |
| ---------------------------------------- | ------------- | ------------ | ----- |
| Accueil / Signature / Events / Education | ✅             | ✅            | ✅     |
| Self Inquiry Tools and Surveys           | ❌             | ✅            | ✅     |
| AdminDashboard / Gestion utilisateurs    | ❌             | ❌            | ✅     |
| Modifier vidéos / textes / fichiers      | ❌             | ❌            | ✅     |

---

### 6. **Flux de Navigation**

```text
Utilisateur non connecté
 └─ /login → AuthContext → Redirection selon rôle

Utilisateur connecté
 ├─ SidebarMenu
 │   ├─ Cliquer sur menu → ContentDisplay met à jour contenu
 │   └─ Permissions appliquées (affichage conditionnel)
 └─ Admin
     ├─ Peut modifier contenus
     └─ Accès AdminDashboard
```
